# GDSC_pick_me
이름을 입력하면 지정한 그룹의 크기만큼 그룹이 나누어 지고
그 그룹에 지정된 테마를 랜덤하게 지정시켜준다

제약사항 : 그룹의 크기는 테마의 개수를 넘을 수 없음
